=== Installazione Casa Vacanza Prenotazioni ===

IMPORTANTE — LEGGERE PRIMA DI INSTALLARE
Se il sito va in errore dopo ogni aggiornamento, segui ESATTAMENTE questa procedura.

PROCEDURA SICURA (consigliata)
1. Vai in Plugin e DISATTIVA "Casa Vacanza Prenotazioni" (se visibile)
2. Nel file manager elimina TUTTE le cartelle che iniziano con:
   casa-vacanza-prenotazioni
   (es. casa-vacanza-prenotazioni, casa-vacanza-prenotazioni-main, casa-vacanza-prenotazioni-old)
3. In WordPress: Plugin → Aggiungi nuovo → Carica plugin
4. Carica il file zip dalla release GitHub (NON estrarre manualmente lo zip)
5. Installa ora → Attiva

METODO CONSIGLIATO — AGGIORNAMENTO DA GITHUB
- WordPress: Plugin → Aggiorna (usa lo zip allegato alla release, non il codice sorgente)
- Oppure, se hai clonato il repo: cd wp-content/plugins/casa-vacanza-prenotazioni && git pull
- Dopo l'aggiornamento la cartella deve chiamarsi esattamente: casa-vacanza-prenotazioni

METODO CONSIGLIATO
- Usa sempre lo zip dalla pagina Releases: tag v1.1.5 o superiore
- NON scaricare il branch "main" come zip da GitHub
- NON usare Plugin → Aggiorna finché non hai una sola cartella pulita

STRUTTURA CORRETTA SUL SERVER
wp-content/plugins/casa-vacanza-prenotazioni/casa-vacanza-prenotazioni.php

ERRORE "Il pacchetto non può essere installato" / AGGIORNAMENTO FALLITO
- NON usare Plugin → Aggiorna se hai cartelle duplicate o installazioni precedenti errate
- Disattiva il plugin (se possibile)
- Elimina TUTTE le cartelle casa-vacanza-prenotazioni* in wp-content/plugins
- Plugin → Aggiungi nuovo → Carica plugin → scegli casa-vacanza-prenotazioni.zip dalla RELEASE GitHub
- NON usare lo zip "Source code" di GitHub né zip creati manualmente dal Mac (contengono .git)
- Attiva il plugin

ERRORE "Il file del plugin non esiste"
- Il database punta a un percorso eliminato
- Elimina le cartelle duplicate e reinstalla lo zip
- Oppure usa tools/cvp-recovery.php nella root WordPress (vedi repository GitHub)

SE IL SITO È IN ERRORE (white screen)
1. Rinomina wp-content/plugins/casa-vacanza-prenotazioni in casa-vacanza-prenotazioni-OFF
2. Il sito tornerà online
3. Reinstalla seguendo la procedura sicura sopra

Release: https://github.com/evolofabio/casa-vacanza-prenotazioni/releases
